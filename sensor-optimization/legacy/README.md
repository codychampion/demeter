# Sensor Optimization Legacy Snapshots

This directory preserves the original repository layouts that fed into the canonical Demeter sensor-optimization module.

## Contents

| Path | Origin | Purpose |
|---|---|---|
| `d2/` | `codychampion/Demeter2` | Full historical snapshot of the larger sensor/filter optimization repository. |
| `sd/` | `codychampion/sensordevlopment` | Full historical snapshot of the smaller early sensor-development branch. |

## How to read this directory

Start with:

- `../R/sensor_optimization.R`
- `../R/simulate.R`
- `../data/README.md`

Use the snapshot directories when you want the original file layouts, raw experiment inputs, or the uncleaned script variants that preceded the canonical historical workflow.
