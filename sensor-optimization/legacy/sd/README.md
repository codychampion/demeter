**https://github.com/codychampion/demeter**

The work in this repo now belongs with the broader Demeter research arc under:

```text
sensor-optimization/
```

## What this repo contains

This repository preserves an earlier subset of the sensor optimization workflow:

- `sensorv3.R`
- `functions.R`
- filter catalog inputs such as `FWHM.csv` and `ranges.csv`
- the Pi camera response curve used by the original experiment

## Canonical location

The maintained historical presentation of this work now lives in:

- `codychampion/demeter`
- `codychampion/demeter/sensor-optimization/R/sensor_optimization.R`
- `codychampion/demeter/sensor-optimization/R/simulate.R`

## Status

This repo should not be featured separately. It is retained only as a historical precursor to the sensor optimization material now consolidated into Demeter.
